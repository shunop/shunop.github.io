  3. 安装centos

本文的侧重点是配置多节点的网络，每个节点都是固定ip。

测试环境：

  • 镜像：CentOS-7-x86_64-DVD-2003.iso
  • 宿主机：macOS Big Sur
  • 软件：VMware Fusion

安装第一个节点

一路默认即可，cpu核心数和宿主机保持一致，网络选 NAT 模式，装的时候选择带桌面环境的GONE。

配置网络

安装好之后在宿主机上查找 NAT 的子网网段

找到下面这一段文字

解释：网关 ip 是 192.168.255.1 ，子网掩码是 255.255.255.0 ，所以我们的节点ip可以设置为 192.168.255.2-192.168.255.255 之间的任何一个。

node1=192.168.255.101；node2=192.168.255.102；

修改或添加这几项 其中 GATEWAY 后面填查出来的网关ip，NETMASK后面填查出来的子网掩码。

保存退出，重启网络服务

检查验证网络

在宿主机打开terminal工具 使用ssh命令连接节点 node1

至此第一个节点配置完成

在虚拟机资源库中右键刚刚的node1点击创建完整克隆就多了一个节点，只需要在里面把网络ip配置好就行了。

保存退出，重启网络服务
